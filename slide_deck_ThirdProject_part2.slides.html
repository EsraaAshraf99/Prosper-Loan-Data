Loan Status prediction¶

by Esraa¶

Investigation Overview¶

In this presentation we will highlight some insights that help us to predict whether the user will be able to complete the loan or not and what are the features that we should take into consideration when taking a decision when some one ask for a loan.

Dataset Overview¶

We will wok on "Prosper Loan Data" data set which contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others.
